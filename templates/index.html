Spam Message Detector

Powered by machine learning — paste any message and know in seconds.

Ham (Safe)

Legitimate messages from real people — personal texts, work chats, and genuine notifications.

Spam (Threat)

Unsolicited junk messages — phishing attempts, scam offers, prize frauds, and promotional spam.

How It Works

Uses TF-IDF text vectorisation + Logistic Regression trained on 5,572 labeled messages.
